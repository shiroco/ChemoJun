ケモじゅんランチャーとは

ケモじゅんランチャーは、 「ケモじゅん」 をインストールすると自動的に開き、 ツールの主な機能へのアクセスを提供します。

画面

画面説明

ケモじゅんランチャーの画面は以下のようになっています。

メニューを選択して、「ケモじゅん」 の機能を利用します。

ウィンドウの上部にあるメニューです。

Molecular Viewer
分子ビューア

分子の構造を３Ｄで表示する 「 分子ビューア 」を開きます。

Molecular Viewer - Select Data
分子ビューア - データ選択

サンプルデータフォルダ内にある分子ファイルを「 分子ビューア 」で参照するには、このメニューが便利です。
フォルダ内にある分子ファイルのリストが出てきます。

分子ファイルがリストされない場合は、データフォルダがないか、環境設定に間違いがあります。
「初期設定」の「分子ファイルのデータ選択」を確認してください。

また、[Shift]キーを押しながらメニューを選択すると、リスト内の分子ファイルを複数選択できます。
いくつかの分子ビューアを一度に開く場合に便利です。

Chemical Media Navigator
化学メディアナビゲータ

分子ファイルなどをすぐに表示できる 「 化学メディアナビゲータ 」を開きます。

Molecular Catalogue
分子カタログ

分子のサムネイルを一覧表示する 「 分子カタログ 」を開きます。

Molecular Catalogue - Select Data
分子カタログ - データ選択

サンプルデータフォルダ内にあるディレクトリを「 分子カタログ 」で一覧するには、このメニューが便利です。
フォルダ内にあるサブディレクトリのリストが出てきます。

ディレクトリがリストされない場合は、データフォルダがないか、環境設定に間違いがあります。
「初期設定」の「分子ファイルのデータ選択」を確認してください。

また、[Shift]キーを押しながらメニューを選択すると、ディレクトリを複数選択できます。
いくつかの分子カタログを一度に開く場合に便利です。
