はじめに

日本語環境を利用できる場合は、オプションで「ケモじゅん」のメニューやメッセージを日本語にすることができます。

日本語環境に必要なものはフリーソフトウェアではありませんので、日本語環境でない方は残念ながら使用することができません。
ご了承ください。

また、これらの解説では、上記の日本語環境に加え、パーセルのインストール方法に沿って「ケモじゅん」が正常にインストールされているものとします。

Windows & Macintosh 版 での説明ですが、 それぞれ違う箇所は、以下のように背景色を変えています。

Windows 版 の説明

Macintosh 版 の説明

ただし、現状、VisualWorks 7.x の Mac OS 9/X では、日本語ロケールが動作しませんので、こちらの日本語辞書も設定しないでください。

準備するもの

以下のものが必要となります。
注：（）内の ??? は、000 もしくはバージョン番号になります。

  • ChemoJun for Smalltalk 環境

    詳細は 「ケモじゅん」インストール で、パーセルのインストール方法を参照してください。

  • 日本語環境

    詳細は 日本語環境 インストール を参照してください。

  • 日本語辞書（ChemoJun???ja.zip）

    「ケモじゅん」で日本語メニューや日本語メッセージを表示する際に必要です。
    また、「じゅん」の日本語辞書も必要です。

日本語辞書の設定

ChemoJun???ja.zip を解凍して、「じゅん」の日本語辞書と同じフォルダへ移動してください。
最終的に、以下のようになっていることを確認しましょう。

$(インストールディレクトリ)
    +- messages
        +- jun
            +- ja
                +- chemoJunDictionary.idx
                +- chemoJunDictionary.lbl
                +- junDictionary.idx
                +- junDictionary.lbl

その他の設定方法は「じゅん」の日本語辞書の設定と同様です。
